# node-farm

## Project Overview

Node-Farm is a simple web project created as part of a learning course on Udemy. The project focuses on building a farm product website using HTML as the frontend and Node.js for the backend. It serves as a hands-on exercise to understand the basics of web development and the integration of server-side scripting with Node.js.

## Key Features

- Responsive HTML template for a farm product website.
- Node.js backend for serving and handling data.
- Learn the fundamentals of routing and handling HTTP requests.
- Gain practical experience in connecting the frontend and backend.

## Technologies Used

- HTML5
- CSS3
- node.js
